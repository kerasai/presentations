Custom Records

Pros

  • Fully flexible and customizable
  • Integrate with 3rd parties

Cons

  • Large amount of effort
  • High potential for bugs or security issues

Piggy-backing

Implementation

  • Custom table in hook_schema()
  • Utilize entity hooks to load and save
  • Alter forms for input
  • Alter entity rendering for display
  • Use "extra" fields to adjust rendering.

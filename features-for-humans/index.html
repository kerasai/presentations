Features-Integrated Modules

Provide components

Features-Integration

  • hook_features_api()
  • hook_features_export()
  • hook_features_export_options()
  • hook_features_export_render()
  • hook_features_revert()
  • hook_features_rebuild()

See features.api.php.

hook_features_api(): Declare component types

hook_features_export(): Declare components to bundle

hook_features_export_options(): Enumerate (list) all instances of component

hook_features_export_render(): Writes PHP code that represents a component

hook_features_revert(): Clears-out overridden config from DB

hook_features_rebuild(): Sets config into database (faux)
